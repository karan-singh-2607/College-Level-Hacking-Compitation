event Description

In this event you will be provided several security oriented challenges for your entertainment you have to cross all the levels in the given time limit. Each team will be given a computer system with a live server. In which you will get information about next level as you crossed the previous level. There will be total 7 levels in this event, the team which is up to the highest level till the last or the team who has crossed all the levels will be winner.
