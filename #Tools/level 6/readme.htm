Service Pack 6 for Visual Basic 6.0, Visual C++ 6.0 and Visual Source Safe 6.0

Service Pack 6 for Visual Basic 6.0, Visual C++ 6.0 and Visual Source Safe 6.0 contains updates to the following programs:

  • Visual Basic 6.0
  • Visual C++ 6.0
  • Visual Source Safe 6.0

This service pack does not contain fixes for the following products, whose mainstream life-cycle has ended:

  • Visual InterDev 6.0
  • Visual FoxPro 6.0
  • Visual J++ 6.0

For more information on product life-cycles, please see the product life-cycle page for Visual Studio. Service Pack 5, which contains fixes for all of the above products, can still be found for a limited time here.

Because system updates have moved to a Windows Update distribution, the following have been removed from Service Pack 6:

  • Microsoft Data Access
  • Microsoft Virtual Machine
  • Microsoft Internet Explorer

To obtain more recent updates for system components, please visit Windows Update.

Changes to Visual Basic 6.0

The primary fixes for Visual Basic concern stability and security enhancements made to the VB runtime. SP6 will help you stay more secure and stable. Most of these fixes are already present in your operating system version of the runtime, such as Windows XP and Windows 2000.

In addition, we have made numerous fixes over the years for customers requesting hotfixes on the VB 6.0 product. A current list of fixes can be found at here.

Changes to Visual C++ 6.0

We've made several improvements to Visual C++ 6.0 by fixing bugs reported to us by customers via the Product Support Services Hotfix process.

In addition, the Visual C++ Processor Pack (VCPP) was removed from Service Pack 6. If you have the VCPP installed, installing SP6 will remove it from your machine. If you wish to continue using the VCPP, you will need to stay with SP5 or migrate to Visual Studio 2002 or 2003 (recommended).

A current list of changes to Visual C++ 6.0 can be found here.

Changes to Visual Source Safe 6.0

This service pack contains the latest Visual Source Safe 6.0 version ("d"). Please note that Visual Source Safe 6.0c was previously released as Visual Source Safe 6.0 Service Pack 6. This new service pack replaces the "c" version of Source Safe with the "d" version found in the latest versions of Visual Studio .NET.

A current list of changes to Visual Source Safe 6.0d can be found here.
